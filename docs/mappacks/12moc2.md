12 Maps Of Christmas 2 is a winter themed platformer based on SMB and the obvbious sequel to [12 Maps Of Christmas 1](https://greatbritdan.github.io/archives/mappacks/12moc1/) released for Alesan99's Entities, Originally released on the 24th of December, 2019. The mappack hasn't aged well but still has some vaugly interesting levels.

## Links

Both the original mappack as well as a fixed port are avaialble to download here. The port only fixes issues caused since release due to AE updates, No content has been changed. I recommend using the fixed port as the original version is here mainly for archival purposes.

Minecraft 2 is a platformer mappack based on Minecraft and the original mappack [Minecraft](https://greatbritdan.github.io/archives/mappacks/minecraft/) and was made for Alesan99's Entities, And is the only unfinished mappack to have been released at some point in time but has since been taken down, the mappack is nothing special and much worse than the original mappack so isn't worth playing today.

## Links

Both the original mappack as well as a fixed port are avaialble to download here. The port only fixes issues caused since release due to AE updates, No content has been changed. I recommend using the fixed port as the original version is here mainly for archival purposes.
